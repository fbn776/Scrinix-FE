## Development

1. Install the dependencies.
   ```bash
    npm install
    # or
    pnpm install
   ```
2. Start the development server.
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
